# Visualizador de Triángulos: Caso LLA (Lado-Lado-Ángulo)

![Versión](https://img.shields.io/badge/versión-1.0.0-blue)
![Licencia](https://img.shields.io/badge/licencia-MIT-green)
![Tecnologías](https://img.shields.io/badge/tech-VanillaJS%20|%20Canvas%20|%20MathJax-orange)

Esta herramienta interactiva permite visualizar y resolver el **caso ambiguo de la trigonometría** (SSA/LLA). El simulador ayuda a comprender gráficamente por qué, dados dos lados y el ángulo opuesto a uno de ellos, pueden existir cero, una o dos soluciones posibles.

## Características Principales

* **Interactividad en Tiempo Real:** Ajuste dinámico del ángulo $$\alpha$$, el lado adyacente $$b$$ y el lado opuesto $$a$$ mediante deslizadores.
* **Motor Gráfico en Canvas:** Renderizado fluido que incluye la representación de la altura ($$h$$), la circunferencia de radio $$a$$ y las rectas base.
* **Análisis Predictivo:** Clasificación automática del estado del triángulo mediante badges de color (0, 1 o 2 soluciones).
* **Precisión Matemática:** Visualización de los cálculos de la altura y las longitudes resultantes para el tercer lado ($$c$$).
